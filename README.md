# Boundless Moments Photography Portfolio

## Overview
**Boundless Moments** is a multi-page photography portfolio website designed to showcase the work of a talented photography team. The website features a modern and elegant design, providing an immersive experience for visitors to explore various photography categories, learn about the team's experience, and get in touch for inquiries.

## Project Structure
The project is organized as follows:

```
boundless-moments
├── index.html
├── portfolio.html
├── gallery.html
├── experience.html
├── contact.html
├── admin.html
├── css
│   └── style.css
├── js
│   └── script.js
├── assets
│   └── fonts
└── README.md
```

## Features
- **Homepage**: A sticky navigation bar, hero section with a large background image, and sections for about, portfolio categories, and website purpose.
- **Portfolio Page**: Showcases the photography team's work with filter buttons for different categories and portfolio cards.
- **Gallery Page**: A visually rich gallery layout displaying high-quality images in a masonry style.
- **Experience Page**: Details the team's skills, experience, and professional strengths with engaging stats.
- **Contact Page**: Provides a contact form for inquiries, with validation and localStorage functionality.
- **Admin Panel**: A demo admin panel for managing inquiries, featuring a login system and a dashboard.

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Responsive Design
- Local Storage for form submissions

## Setup Instructions
1. Clone the repository or download the project files.
2. Open the `index.html` file in a web browser to view the homepage.
3. Navigate through the different pages using the navigation bar.
